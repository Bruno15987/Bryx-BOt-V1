## 🚀 Instalação via Termux

Para instalar o BRYX-BOT V1.0 no Termux, siga os passos abaixo:

### Pré-requisitos
Certifique-se de que seu Termux esteja atualizado e com as dependências necessárias. Será solicitado que você confirme a instalação digitando `y` em alguns momentos.

### Comandos de Instalação

**1. Atualizar pacotes e instalar dependências:**
```bash
apt-get update -y && pkg upgrade -y && pkg update -y && pkg install nodejs -y && pkg install nodejs-lts -y && pkg install ffmpeg -y && pkg install wget -y && pkg install tesseract -y && pkg install git -y
```

**2. Conceder permissão de armazenamento:**
```bash
termux-setup-storage
```

**3. Clonar o repositório e navegar até a pasta:**
```bash
cd /sdcard/Download && git clone https://github.com/Bruno15987/Bryx-BOt-V1 && cd /sdcard/Download/Bryx-BOt-V1
```

**4. Instalar as dependências do projeto:**
```bash
npm install --force --no-bin-links
```

## 💡 Uso

Após a instalação, você pode iniciar o bot com o seguinte comando:

```bash
npm start
```

Siga as instruções no terminal para configurar e usar o bot.
